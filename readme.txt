Hug post view counter calculate how many times a post,  custom-post-type had been viewed with simply method.

== Description ==
Counting  method:
-  30 seconds actived.
-  2/3 post\'s content had been scrolled.

Just  active and use. No configuration needed.

== Installation ==
1. Install Hug Post View Counter either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the ‘Plugins’ menu in WordPress
